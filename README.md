# Light Maps Plus

Lightweight Flutter maps application with navigation, Bluetooth music management, and settings — optimized for low-performance devices.

## Tech Stack

- **Maps**: OpenStreetMap (OSM) via `flutter_map`
- **Navigation**: OSRM (Open Source Routing Machine)
- **Bluetooth**: Flutter Bluetooth for music device management
- **Framework**: Flutter / Dart

## Features

### Maps
- [ ] Display interactive OSM map
- [ ] Current location tracking
- [ ] Map zoom / pan controls
- [ ] Place search / geocoding
- [ ] Bookmarked locations
- [ ] Offline maps

### Navigation
- [ ] Route calculation via OSRM
- [ ] Turn-by-turn directions
- [ ] Route preview on map
- [ ] Alternative routes
- [ ] ETA and distance display
- [ ] Voice navigation prompts

### Bluetooth Music Management
- [ ] Scan and pair Bluetooth audio devices
- [ ] Display connected device info
- [ ] Media playback controls (play, pause, skip)
- [ ] Volume control
- [ ] Auto-reconnect to known devices

### Settings
- [ ] Map style / tile source selection
- [ ] Navigation preferences (avoid tolls, shortest vs fastest)
- [ ] Bluetooth device management
- [ ] Units (km / mi)
- [ ] Theme (light / dark)
- [ ] Language selection
- [ ] Cache / offline map settings
